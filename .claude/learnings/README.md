# Learnings

This directory contains accumulated knowledge from building and maintaining this project.
Each file covers a specific topic. Claude Code loads these automatically via CLAUDE.md references.

## Files

- `integrations.md` — Patterns for adding new integrations (MCP servers, skills, APIs)
- `deployment.md` — Deployment, systemd, and infrastructure lessons
- `sdk.md` — Claude Agent SDK usage patterns and gotchas
- `testing.md` — Testing patterns and common pitfalls
